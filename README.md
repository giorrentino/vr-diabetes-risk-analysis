# VR Diabetes Risk Assessment System

This project is a Virtual Reality application developed in Unity for the simulation of diabetes risk assessment in a clinical environment.

## 📌 Overview

The system allows users to interact with virtual patients and analyze clinical data in an immersive environment.  
It is based on real-world epidemiological data (NHANES) and integrates Machine Learning concepts for decision support.

## 🧠 Key Features

- VR environment developed with Unity and XR Interaction Toolkit
- Interactive patient simulation
- Clinical data visualization in VR
- Decision-making experience based on real data

## 📊 Dataset

The application is based on the NHANES dataset (National Health and Nutrition Examination Survey), used during the data analysis and model development phase.

## 🧪 Machine Learning

Machine Learning models (Random Forest, XGBoost) were developed separately for diabetes risk classification and used to guide the logic of the application.

## ⚙️ Technologies Used

- Unity (C#)
- XR Interaction Toolkit
- Python (for data analysis and ML, external to this repository)
- Pandas, Scikit-learn, XGBoost
- Git & GitHub

## 🕶️ VR Application

- Patient spawning system
- Interactive clinical records
- User interaction via XR controllers
- Feedback system based on user decisions

## 🎯 Objectives

- Combine real-world data with immersive environments
- Improve understanding of diabetes risk factors
- Explore VR as a tool for education and simulation

## 📌 Notes

The Machine Learning models and data preprocessing were developed separately and are not included in this repository.
